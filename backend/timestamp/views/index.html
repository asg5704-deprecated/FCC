Timestamp Microservice

built with Moment.js

How to Use:

Add a date in the URL address bar, e.g. December 29, 2018

https://fcc-backend-01.glitch.me/December 29, 2018 OR

You can add a date in Unix format, e.g. 1520182029

https://fcc-backend-01.glitch.me/1450137600

Output:

{unix: "115223223", humanDate: "December 29, 1988}
